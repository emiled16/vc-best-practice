# Notion MCP

Use the Notion MCP server for live workspace operations.

## Workflow

1. Search or fetch the relevant page, database, or data source first.
2. Read the current structure before editing.
3. Use precise updates instead of broad replacements when modifying content.
4. Verify page IDs, data source IDs, and property names before writing.

## Good Fits

- Search workspace content.
- Read and summarize live pages or databases.
- Create or update pages, comments, and database rows.
- Query data sources and meeting notes.

## Guardrails

- Prefer incremental edits over whole-page replacement.
- Confirm the exact target before changing workspace content.
- Use the current schema and page structure instead of guessing property names.
